# Автоматизация инфраструктуры в Yandex Cloud через Terraform и Ansible

Проект автоматического развертывания трех виртуальных машин (Ubuntu 22.04 LTS) в облаке с последующей конфигурацией веб-серверов и балансировкой трафика по алгоритму Round Robin.

## Компоненты проекта
* **Terraform**: Отвечает за IaC (Infrastructure as Code) — автоматическое создание облачной сети, подсети и 3 инстансов ВМ.
* **Ansible**: Отвечает за управление конфигурацией:
  * Настройка уникальных системных `hostname` из инвентаря.
  * Установка базового набора утилит (`net-tools`, `dnsutils`, `wget`, `curl`, `mc`, `rsync`).
  * Развертывание бэкенд страниц `nginx_page` на `vm-2` и `vm-3`.
  * Развертывание реверс-прокси балансировщика `nginx_proxy` на `vm-1` (порт 3000).

## Как запустить
1. Развернуть ресурсы:
   ```bash
   terraform init
   terraform apply
   ```
2. Настроить конфигурацию ОС и веб-серверов:
   ```bash
   cd ansible
   ansible-playbook playbook.yml
   ```
